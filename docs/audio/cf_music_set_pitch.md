Sets the pitch for the music.

```cpp
void cf_music_set_pitch(float pitch);
```

Parameters | Description
--- | ---
pitch | The pitch to set, default to 1.0f.

## Remarks

This is a playback speed multiplier, meaning negative numbers will play the music backwards,
while positive numbers play forwards.
